🚀 Erste Schritte

Was ist PEAKING?

PEAKING ist dein Solo-Creator-Tool für Instagram-Wachstum — Reel-Tracking, AI-Brain für Strategy-Memos, Auto-Edit-Assistant für Captions, und vieles mehr.

Built solo. Ohne VC. Ohne Hype. Always peaking.

Wie melde ich mich an?
  1. 1peaking.world öffnen → Login-Screen
  2. 2Sebi-Login-PW eingeben (24h Session via SHA-256)
  3. 3Dashboard ist offen → Account-Switcher oben rechts wählen
Account-Switch — wofür?

Du hast 2 Insta-Accounts: @vegetarianhulk (🏔 Plant-Based) + @peakingworld (🚀 Builder-Tool). PEAKING speichert Daten pro Account getrennt — Reels, Ideen, Plan, Stats. Switch → Dashboard zeigt nur den gewählten Account.

Dashboard-Stats verstehen
  • Follower: Live von Meta API (alle 6h gesynct)
  • Diese Woche: Reels mit Timestamp < 7 Tage
  • Streak: Consecutive Tage mit ≥1 Post
  • Total Reels: alle gecachten Reels für den Account

🟢 Grüner Punkt neben Label = Live-Data aus Worker-Cache

📊 Reel-Tracker

Wie nutze ich den Auto-Tracker?

Der Auto-Tracker zeigt deine Reels automatisch aus der Meta API. Kein Screenshot-Upload mehr nötig!

  1. 1Brand-Tab wählen (Alle / Peaking / Hulk / Smash)
  2. 2Sortieren nach Datum / Likes / Comments
  3. 3Auf Reel klicken → öffnet Insta-Permalink
  4. 4Refresh-Button drücken für sofortiges Sync
Was zeigt die Sparkline?

Die kleine Linie unter jedem Reel zeigt den Likes-Trend über die letzten Sync-Cycles. Grünes Delta = Wachstum, rotes Delta = stagniert.

⚠️ Braucht mindestens 2 Sync-Cycles (12h+) für Daten.

Sync-Cycle: wie oft?

Alle 6 Stunden automatisch via Cloudflare-Cron. Plus: Manual-Refresh-Button für sofortiges Sync.

Welche Daten kommen wann?
Likes + Comments✅ JETZT
Caption + Thumbnail + Permalink✅ JETZT
Reach + Impressions + Plays🟡 nach Meta App-Review
Saves + Shares + Watch-Time🟡 nach Meta App-Review

🧠 AI Brain

Was ist AI Brain?

Alle Claude-Memos der Sessions — sortiert nach Brand (PEAKING/SMASH/hulk/Cross), Type (project/feedback/learning/reference), und Status (GO/PARK/DONE).

Plus: Live-Search, Timeline-View, Memo-Detail-Drawer mit Cross-Refs.

Wie filter ich?
  • Brand-Pills: Alle / 🚀 PEAKING / 💪 SMASH / 🌱 hulk / 🔗 Cross
  • Type-Pills: Project / Feedback / Learning / Reference / User
  • Status-Pills: GO / PARK / DONE / LIVE / BLOCKED
  • Search-Input: free-text in Title + Description
Timeline-View?

Toggle „📅 Timeline" oben → Memos gruppiert nach Modified-Date (Heute / Gestern / Wochentag-Datum).

🛠 Troubleshooting

„Session has expired" / Token-Fehler

Der Meta-Token ist abgelaufen (alle 60 Tage). Token-Refresh:

bash /tmp/peaking-token-convert.sh

Bald: Auto-Refresh-Cron im Worker (nach META_APP_SECRET-Setup). Dann nie wieder manuell.

Stats zeigen falsche Zahlen
  1. 1. Hard-Reload: Cmd+Shift+R
  2. 2. Service Worker leeren: DevTools → Application → SW → Unregister
  3. 3. Refresh-Button im Live Insta Sync
Account-Switch zeigt alte Daten

Page-Reload nach Switch ist normal. Wenn Daten trotzdem nicht updaten: Cache leeren (siehe oben).

Reel-Edit: „File zu groß"

Max 24 MB (KV-Limit). Workaround: MP3-Bitrate auf 96kbps reduzieren (reicht für ~30 Min). In DaVinci: Audio-Render-Settings.

Reel-Edit: „REPLICATE error"

Whisper-Service kurz down. In 1-2 Min nochmal probieren.
