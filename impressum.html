KI-gestützte Inhalte (EU AI-Act Art. 50, ab 02/2026)

PEAKING ist ein KI-gestütztes Tool. Folgende AI-Services kommen zum Einsatz:

  • Anthropic Claude API — für Text-Generierung, Strategie-Analysen, Hook-Vorschläge
  • Replicate (Flux Dev, Recraft V3, Whisper) — für Bild-Generierung, Untertitel-Extraktion
  • Meta Graph API — für Instagram-Stats-Sync (eigene Konten des angemeldeten Users)

Generierte Inhalte werden klar als solche erkennbar gemacht (Watermark, Disclaimer). Der angemeldete User behält die volle Kontrolle: er kann generierte Inhalte vor der Veröffentlichung prüfen, anpassen oder verwerfen.
